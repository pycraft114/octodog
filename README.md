# 화이팅

### 코드 컨벤션

- HTML의 id, class는 lower case에 (-)를 사용한다.
- 이슈의 완료 조건은 체크 박스로 작성
- 네임 스페이스 단위로 함수와 변수를 작성한다.

### 작업의 프로세스

1. 함수 단위로 커밋을 자주 한다
2. 커밋할 때마다 이슈에 완료 조건 생성/체크를 지향한다.
3. Push후 이슈가 완료되었다면 프로젝트에서 이슈 카드를 Done에 위치시킨다.
4. 다음 작업할 이슈를 Doing으로 가져온다.
5. 작업이 끝날때 까지 무한 반복
